# BatRun-plugins
Centralizing some of my plugins for management via BatRun.

**RetroBatGameMode**

- At startup, reduces the memory consumption of certain processes not protected by the system. This allows your applications to remain inactive and be reactivated after RetroBat closes.
The plugin will only work when RetroBat is launched and remains active. 
For example, if, like me, you use Chrome with many tabs and it consumes gigabytes of memory, its consumption will be reduced to almost zero, and it will be reactivated when RetroBat closes.
Windows Explorer is hidden rather than stopped. Unlike Batrun's Shell mode, this method does not prevent access to software that depends on the explorer.exe process, nor to the Windows Game Bar. If a suspended process is causing problems, you can add it to the whitelist.
___________

**AttractMode**

- It navigates autonomously through RetroBat's various game systems, randomly selecting a game. It does not launch the games; it simply selects one for a few seconds before moving on to the next.
